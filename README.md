An educational space adventure game where children travel between planets answering trivia questions powered by AI to refuel their rocket ship.

## Features

- **AI-Powered Trivia**: Infinite questions generated by Google Gemini API.
- **Adaptive Difficulty**: Difficulty levels for different age groups (Cadet, Explorer, Ace).
- **Interactive Space Travel**: Fun animations, warp speed effects, and planet exploration.
- **Captain's Log**: A digital passport system to collect badges and track progress.
- **Progress Tracking**: Fuel management and score persistence.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Set up Environment Variables**
   Create a `.env` file in the root directory:
   ```
   API_KEY=your_google_gemini_api_key
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   ```

## Deployment on Vercel

1. **Push to GitHub/GitLab/Bitbucket**: Push your code to a repository.
2. **Import Project**: Go to [Vercel](https://vercel.com), click "Add New", and import your repository.
3. **Configure Project**:
   - **Framework Preset**: Vercel should automatically detect `Vite`.
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
4. **Environment Variables (CRITICAL)**:
   - Expand the **Environment Variables** section.
   - Key: `API_KEY`
   - Value: `Your Actual Google Gemini API Key` (starts with `AIza...`)
   - Click **Add**.
5. **Deploy**: Click "Deploy".
